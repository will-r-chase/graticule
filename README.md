# Graticule

Graticule is an open-source, browser-based map-making tool. Import geographic data, style it, choose a projection, and export your map — no GIS software required.

## Announcement

Read the [introductory blog post](https://www.williamrchase.com/writing/2026-06-03-introducing-graticule) for background on what Graticule is and why I built it.

## Reporting issues

Found a bug or have a feature request? [Open an issue](https://github.com/will-r-chase/graticule/issues) on GitHub. Please include steps to reproduce the problem and, if relevant, a screenshot or sample file.
