title: Canvas Modes
description: The four ways to interact with the map canvas in Graticule — Pan, Select, Edit, and Draw.
---

The map canvas has four modes that you can swap between using the floating toolbar on the canvas, or with a keyboard shortcut.

| Mode       | Shortcut | What it's for                  |
| ---------- | -------- | ------------------------------ |
| **Pan**    | `V`      | Navigating the map             |
| **Select** | `S`      | Selecting layers and features  |
| **Edit**   | `E`      | Reshaping a feature's vertices |
| **Draw**   | `D`      | Creating new features          |

While in any mode other than Pan, hold `Space` to pan temporarily without switching modes.

## Pan mode

Drag to pan a flat map, or drag to rotate the globe. 

## Select mode

Click, shift-click, or drag a marquee to select layers, or double-click into a layer to select its features. Once something is selected, an action bar offers operations that depend on the selection.

For the full selection model, see [Selecting layers](/layers/working-with-layers/#selecting-layers) and [Working with Features](/features/working-with-features/).

## Edit mode

Reshape a feature by dragging its vertices directly on the canvas, and add or remove vertices as needed.

For the full workflow see [Editing Features](/features/editing-features/).

## Draw mode

Create new point, line, and polygon features by drawing them on the canvas. See [Drawing](/canvas-modes/drawing/).
